# 字体颜色大小样式

<br>

```java
//size：规定文本的尺寸大小。可能的值：从 1 到 7 的数字。浏览器默认值是 3
//face：规定是什么字体
<font color=#A52A2A size=4 >Markdwon测试</font>
<font color=#D2691E size=4>**Markdwon加粗测试**</font>
<font face=STCAIYUN>我是华文彩云</font>
<span style="color:red">this text is red</span>
```
